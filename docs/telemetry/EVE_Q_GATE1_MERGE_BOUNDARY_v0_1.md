# EVE_Q++ Gate 1 Merge Boundary v0.1

Merging pilot infrastructure means the repository can build and test a read-only observation membrane.

It does not mean:

- Gate 1 is active;
- a live source is approved;
- live data may generate proposals;
- any execution capability exists;
- any capital capability exists.

Gate 1 activation remains a later human-promoted transition backed by live-read-only soak, threat-model, and rollback evidence.
